1. FIND A GRADING SCALE.

The most common grading scale for schools in the United States is a four-point scale. Using this scale, a grade of A = 4 points, B = 3 points, C = 2 points, D = 1 point, and F = 0 points. This is called an unweighted GPA.[2] Some schools use a weighted GPA, which allots 5 points for more rigorous classes, such as honors, Advanced Placement (AP), and International Baccalaureate (IB). Other classes are weighted the same. Students who take the 5-point classes can end up with GPAs above a 4.0.
